Como Instalar

Antes de começar a ler os materiais (Instalar o Tensorflow)

Como eu fiz:


No terminal do Ubuntu
  1. Baixar pip.Usar esse comando para python 2.7
    sudo apt-get install python-pip python-dev
    Usar esse comando para python 3
    sudo apt-get install python3-pip python3-dev

  2. Instalar o tensorflow via pip
    Para python 2.7
    pip install tensorflow
    Para python 3
    pip3 install tensorflow
DENTRO DO PYTHON, TESTAR O TENSORFLOW
PARA ENTRAR NO PYTHON: DIGITE PYTHON NO TERMINAL DO UBUNTU, DEPOIS
import tensorflow as tf hello = tf.constant('Hello, TensorFlow!') sess = tf.Session() print(sess.run(hello))
Se exibir Hello, TensorFlow, foi instalado corretamente!
