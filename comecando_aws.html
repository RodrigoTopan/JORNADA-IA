Resumo do Guia de ML da Amazon Web Services


Para que serve

Com o Amazon Machine Learning(Amazon ML), você pode criar e treinar modelos preditivos e hospedar seus aplicativos em uma solução em nuvem escalável em apenas 4 etapas:

  • Criar fonte de dados
  • Criar fonte de dados de treinamento
  • Criar um modelo de ml
  • Usar o modelo para gerar previsões

Criar Fonte de dados

Em aprendizagem de máquina, você geralmente obtém os dados e garantem o formato adequado antes de treinar o seu modelo.

  • Fazer download do dataset
    • https://s3.amazonaws.com/aml-sampledata/banking.csv
    salvar como .csv
  • Fazer o download do arquivo que usaremos para o teste do modelo
    • https://s3.amazonaws.com/aml-sample-data/banking-batch.csv
    Salve-o com o nome banking-batch.csv

Você quer que o modelo de ML responda à pergunta "O cliente se inscreverá no meu novo produto?". No conjunto de dados banking.csv, a resposta a essa pergunta é o atributo y, que contém os valores 1 (para sim) ou 0 (para nenhum). O atributo que você deseja que o Amazon ML saiba como prever é chamado de atributo de destino.

Em seguida, faça upload dos arquivos banking.csv e banking-batch.csv para o Amazon S3. Para fazer upload dos arquivos para um local do Amazon S3

  • 1. Faça login no Console de gerenciamento da AWS e abra o console da Amazon S3 em https:// console.aws.amazon.com/s3/.
  • 2. Na lista All Buckets, crie um bucket ou escolha o local onde você deseja fazer upload dos arquivos.
  • 3. Na barra de navegação, escolha Upload.
  • 4. Escolha Adicionar arquivos.
  • 5. Na caixa de diálogo, navegue até a área de trabalho, escolha banking.csv e bankingbatch.csv; por fim, escolha Open.

Etapa 2: Criar uma fonte de dados de treinamento

Para isso, basta você ir no seu console aws, procurar o serviço Machine Learninng, acionar a opção criar novo datasource e novo modelo, informar onde está armazenado a sua fonte de dados (no seu bucket).
Após realizar o processo, você pode pedir que a aws faça uma inferência dos dados que ela coletou do seu bucket. Uma inferência significa verificar os tipos e nomes dos atributos do banco de dados.

Etapa 3: criar um modelo de ML
O modelo de ML é um conjunto de padrões que o Amazon ML encontra nos dados durante o treinamento. Você usa o modelo para criar previsões.

Para criar um modelo de ML

  • 1. Como o assistente Get started cria uma fonte de dados de treinamento e um modelo, o Amazon Machine Learning (Amazon ML) usa automaticamente a fonte de dados de treinamento que você acabou de criar e leva você diretamente para a página ML model settings.
    Na página ML model settings, em ML model name, verifique se o ML model: Banking Data 1 padrão é exibido. Usar um nome amigável, como o padrão, ajuda você a identificar e gerenciar facilmente o modelo de ML.
  • 2. Em Training and evaluation settings, verifique se Default está selecionado.
  • 3. Em Name this evaluation, aceite o padrão, Evaluation: ML model: Banking Data 1.
  • 4. Escolha Review, revise as configurações e depois escolha Finish.

Depois que você escolheu Finish, o Amazon ML adiciona o modelo à fila de processamento. Quando o Amazon ML cria o modelo, ele aplica os valores padrão e realiza as seguintes ações:
• Divide a fonte de dados de treinamento em duas seções, uma contendo 70% dos dados e outra contendo os 30% restantes
• Treina o modelo de ML na seção que contém 70% dos dados de entrada
• Avalia o modelo usando os 30% restantes dos dados de entrada

Etapa 5: Usar o modelo de ML para gerar previsões
Sim, eu pulei a etapa 4 do manual da Amazon.

Uma previsão em tempo real é uma previsão para uma única observação que gera o Amazon ML sob demanda. As previsões em tempo real são ideais para aplicativos móveis, sites e outros aplicativos que precisam usar resultados interativamente.

  • 1. No painel de navegação ML model report, escolha Try real-time predictions.
  • 2. Escolha Paste a record.
  • 3. Na caixa de diálogo Paste a record, cole a seguinte observação.
  • 4. Na caixa de diálogo Paste a record, escolha Submit para confirmar que você deseja gerar uma previsão para essa observação. O Amazon ML preenche os valores no formulário de previsão em tempo real.
  • 5. Na parte inferior da página, escolha Create prediction.
    • A previsão aparece no painel Prediction results à direita. Essa previsão tem um rótulo previsto especificado em Predicted label como 0, o que significa que esse cliente potencial dificilmente responderá à campanha. Um Predicted label igual a 1 significa que o cliente provavelmente responderá à campanha.
