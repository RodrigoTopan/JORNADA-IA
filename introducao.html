Introdução Machine Learning

Machine Learning


Machine Learning é uma abordagem algoritmica que permite, através de modelos matemáticos e estatísticos, responder questões específicas baseadas em um estudo de dados que abrange o domínio do problema.

Arthur Samuel(1959). Define aprendizado de máquina como o campo de estudo que permite aos computadores aprenderem sem serem explicitamente programados para tal. Arthur Samuel programou uma máquina para jogar xadrez. Ele nunca foi muito bom em xadrez, mas a máquina era pior. Então Arthur fez a máquina jogar dezenas de milhões de vezes contra ela mesma, o que fez com que ela ganhasse muita experiência em xadrez e, a saber, calcular onde era posições boas para as peças e onde eram as posições ruins. Tal fato, fez com que a máquina superasse as habilidades de Arthur em pouco tempo no xadrez.

Outra definição seria a de que machine learning é a capacidade de um programa de realizar determinada tarefa sendo treinado para tal através da repetição, com determinada precisão.

Tipos de Aprendizado


Existem dois tipos principais de algoritmos de ML. Algoritmos de Aprendizado Supervisionado e de Aprendizado não Supervisionado

O termo aprendizado supervisionado se refere ao fato de darmos a um algoritmo um dataset (conjunto de dados) das respostas certas, para que este aprenda e reproduza mais respostas certas.
Em aprendizado supervisionado, nós damos um dataset e nós já temos uma ideia do que esperar de saída, tendo a ideia de que existe um relacionamento entre as entradas e as saídas.
Para o problema de regressão, é tentado predizer uma saída constante. Parao os problemas de classificação, tentamos predizer um valor discreto, verdadeiro ou falso, 0 ou 1.

O que é TensorFlow

Definição


Basicamente, TensorFlow é uma biblioteca de software livre para computação matemática usando dados de fluxos de grafos. TensorFlow é a biblioteca mais popular de aprendizado de máquina atualmente. Os nós dos grafos representam operações matemáticas, enquanto as arestas representam matrizes multidimensionais (tensors) permitindo comunicação entre eles.

Pra que serve?


Tensorflow permite implementar computação em uma ou mais CPUs ou GPUs em um desktop ou vários, servidores ou até mesmo dispositivo mobile com apenas uma única API.
TensorFlow fornece um conjunto extenso de funções e classes que permitem aos usuários definir modelos a partir do zero matematicamente. Isso permite que os usuários com o conhecimento técnico apropriado criem modelos personalizados e flexíveis de forma rápida e intuitiva. Além disso, embora TensorFlow tenha suporte extensivo para a funcionalidade específica de Machine Learning, é igualmente adequado para executar cálculos matemáticos complexos.
TensorFlow foi originalmente desenvolvido por pesquisadores e engenheiros trabalhando na equipe do Google Brain com a pesquisa de inteligência de máquina do google, com o propósito de conduzir pesquisas em machine learning e redes neurais profundas, mas o sistema é geral o suficiente para ser aplicável em uma imensa varíavel de outros domínios também.
